Chaotic API

This is a simple API for resilience tests purposes

Concept

The main idea is to provide an endpoint that randomly presents some kind of problem, it could be a dealy or an internal server error

How-to

By doing a GET HTTP request to https://chaotic-api.herokuapp.com/tryme, you may receive one of the responses below:

  • SUCCESS - { "statusCode": 200, "message": "All good." }
  • DELAY - { "statusCode": 200, "message": "Your response has been delayed." }
  • ERROR - { "statusCode": 500, "message": "Something went wrong." }
